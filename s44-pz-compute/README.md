# BRA-LIN-S4.2 - PZ Tables as Federated Datasets 

Periodic delivery of PZ estimates for all objects in LSST Object catalog, for each data release, using a PZ algorithm to be determined by the recipienr group (Rubin DM Photo-z Coordination), as federated dataset (to be incorporated into Butler datasets available on RSP). 

See the [BRA-LIN in-kind program documentation](https://linea-it.github.io/pz-lsst-inkind-doc/s4_4/) for more details.
